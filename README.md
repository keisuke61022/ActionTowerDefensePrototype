# Action Tower Defense Prototype

## Unityでの確認手順
1. Unity 6.4でこのプロジェクトを開く
2. 任意のSceneを開く（この実装は再生時に自動でゲームオブジェクト生成）
3. Playボタンを押す
4. そのままCPU戦プロトタイプが開始

## 操作方法
- 移動: `WASD` または `矢印キー` または 左下バーチャルスティック
- 射撃: `Space` / 左クリック / 右下 `Shoot`
- ユニット1設置(砲台): `E` / 右下 `Unit1`

## 実装済み内容
- ゲーム開始時に `GameManager` が戦場・拠点・プレイヤー・UIを自動生成
- 敵5ウェーブを自動生成
- プレイヤー射撃と砲台の自動攻撃
- コストシステム（初期5、最大10、設置3、撃破+1）
- 勝利/敗北表示（敵拠点破壊 or 5ウェーブ生存 / 自拠点破壊）
