# Action Tower Defense Prototype

## Unityでの確認手順
1. Unity 6.4でこのプロジェクトを開く
2. 任意のSceneを開く（再生時にゲームオブジェクトを自動生成）
3. Playボタンを押す
4. CPU対戦風プロトタイプが開始

## 操作方法
- 移動: `WASD` / `矢印キー` / 左下バーチャルスティック
- 射撃: `Space` / 左クリック / 右下 `射撃`
- 砲台設置(3コスト): `E` / 右下 `砲台`

## 今回の修正内容（疑似PVP化）
- **敵プレイヤー役 `EnemyCommander`** を追加（上側で行動、赤系カラー）
- 敵キャラが上半分を中心に移動し、近づかれたら距離を取る簡易AIを追加
- 敵キャラが定期的に弾を発射（プレイヤー/味方ユニット/自軍拠点にダメージ）
- **敵コスト（最大10）** を追加し、プレイヤー同様に時間回復
- プレイヤーコストも時間回復を基本化（撃破時ボーナスは残存）
- 敵CPUがコスト3を消費して敵ユニットを配置（2〜4秒CD、配置数上限あり）
- 味方/敵ユニットの色分けと相互衝突ダメージを追加
- プレイヤー弾/敵弾を色分け（青系/赤系）
- 拠点HPを増加し、30〜60秒程度継続しやすい対戦テンポに調整
- UIステータスへ `敵コスト x/10` を追加（デバッグ確認用）

## 現在のゲームループ
- プレイヤー側: 自キャラ操作 + 射撃 + コストで砲台/味方ユニット展開
- 敵CPU側: 敵キャラAI + 裏コスト消費で敵ユニット展開
- 両陣営ユニットが前進し衝突
- 敵拠点破壊で勝利 / 自軍拠点破壊で敗北
